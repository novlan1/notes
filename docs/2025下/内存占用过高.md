## 1. 找到内存占用最高的10个进程

```bash
ps aux --sort=-%mem | head -10
```

## 2. 杀掉匹配完整命令行的进程

```bash
pkill -f "命令"
```

## 3. pm2 logs 指定行数

```bash
pm2 logs my-app --lines 50
```

## 7. 修改 git remote 默认远程地址

```bash
# 先看看当前配置
git config --list | grep remote

# 修改
git config branch.master.remote group
```

`master` 是分支名，不同项目不一样。
