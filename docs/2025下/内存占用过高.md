## 1. 找到内存占用最高的10个进程

```bash
ps aux --sort=-%mem | head -10
```

## 2. 杀掉匹配完整命令行的进程

```bash
pkill -f "命令"
```

## 3. pm2 logs 指定行数

```bash
pm2 logs my-app --lines 50
```
