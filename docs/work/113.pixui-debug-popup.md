# PixUI 调试工具

已实现 PixUI 项目中的调试工具，支持：

- 打印用户 UID
- 打印版本发布时间
- 打印最后提交信息
- 支持复制信息
- 支持区分环境，正式环境不展示
- 支持手动销毁
- 支持自定义信息

<img src="https://mike-1255355338.cos.ap-guangzhou.myqcloud.com/article/2025/12/own_mike_FehZJJ4zfkfGw4AN.png" width="500" />

接入方式

```tsx
// pages.tsx
import { AppWithDebug } from '@tencent/press-pix/debug-popup/app-with-debug';

export const Pages = () => (
  <AppWithDebug vconsoleEnabled={true}>
    <Router history={router}>
      <Route
        path="/"
        component={Home}
      />
      <Route
        path="/room"
        component={Room}
      />
    </Router>
  </AppWithDebug>
);
```
