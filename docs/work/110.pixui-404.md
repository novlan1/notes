# PixUI 开发者工具 404

PixUI 开发者工具打不开页面，显示 404，如下图。

<img src="https://mike-1255355338.cos.ap-guangzhou.myqcloud.com/article/2025/12/own_mike_JzPK2JPtGCaTKkyp.png" width="500" />

复制地址到浏览器一样打不开，如下图。

<img src="https://mike-1255355338.cos.ap-guangzhou.myqcloud.com/article/2025/12/own_mike_xZymkwindN3NyABz.png" width="500" />

解决方案，用本机 `ip` 代替上面的 `ip`，如 `http://10.20.68.15:19916/app/index.html`。

获取本地 IP 地址：

```bash
ipconfig getifaddr $(route -n get default | grep interface | awk '{print $2}')
```
