# 小程序发布 MCP

实现了小程序发布的MCP，可以更丝滑的发布。

## 1. 配置方式

1. 去研发平台申请 `token`，申请地址 https://mobile.woa.com/rd-platform-web/#/management/token
2. 在 CodeBuddy 等工具下添加 mp-publish-mcp“ 如下，`rtoken` 就是上面申请的 `token`

```json
{
  "mcpServers": {
    "mp-publish-mcp": {
      "command": "npx",
      "args": [
        "-y",
        "@tencent/mp-publish-mcp@latest",
        "--rid=guowangyang",
        "--rtoken=xx",
        "--stdio"
      ],
      "env": {
        "NO_PROXY": "mobile.woa.com,git.woa.com"
      }
    }
  }
}
```

## 2. 使用

可以在对话框中输入 **使用 mcp 发布小程序，子工程 gp，发布原因：测试** 等内容进行上传
