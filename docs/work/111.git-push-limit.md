# Git 推送限制

有时，代码推送会报错 `You can't push commit[6ed44986] with committer[xxx] or email[xxx@xxx.com] that does not belong to the account[xxx]`，如下图。

<img src="https://mike-1255355338.cos.ap-guangzhou.myqcloud.com/article/2025/12/own_mike_4Zyi8mXtDNddNSeS.png" width="700" />

解决方案如下，在工蜂、仓库设置、版本设置中，修改“推送限制”为无限制。
