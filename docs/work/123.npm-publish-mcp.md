# NPM 发布 MCP

实现了 NPM 发布的MCP，可以更丝滑的发布。

配置方式

1. 去研发平台申请 `token`，申请地址 https://mobile.woa.com/rd-platform-web/#/management/token
2. 在 CodeBuddy 等工具下添加 ”upload-mcp“ 如下，`rtoken` 就是上面申请的 `token`

```json
{
  "mcpServers": {
    "npm-publish-mcp": {
      "command": "npx",
      "args": [
        "-y",
        "@tencent/upload-mcp@latest",
        "--rid=guowangyang",
        "--rtoken=xx",
        "--stdio"
      ],
      "env": {
        "NO_PROXY": "mobile.woa.com,git.woa.com"
      }
    }
  }
}
```

3. 使用。可以在对话框中输入 **使用 mcp 发布 beta 版本** 等内容进行上传
