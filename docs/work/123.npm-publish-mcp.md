# NPM 发布 MCP

实现了 NPM 发布的MCP，可以更丝滑的发布。

## 1. 配置方式

1. 去研发平台申请 `token`，申请地址 https://mobile.woa.com/rd-platform-web/#/management/token
2. 在 CodeBuddy 等工具下添加 npm-publish-mcp“ 如下，`rtoken` 就是上面申请的 `token`

```json
{
  "mcpServers": {
    "npm-publish-mcp": {
      "command": "npx",
      "args": [
        "-y",
        "@tencent/npm-publish-mcp@latest",
        "--rid=guowangyang",
        "--rtoken=xx",
        "--stdio"
      ],
      "env": {
        "NO_PROXY": "mobile.woa.com,git.woa.com"
      }
    }
  }
}
```

## 2. 使用

可以在对话框中输入 **使用 mcp 发布 beta 版本** 等内容进行上传

<img src="https://cdn.uwayfly.com/article/2026/1/own_mike_jfNyjb3nB6GkDS8S.gif" width="400" />

monorepo 仓库下发布某个子包时，可以输入 **使用 mcp 发布 @tencent/project-config-vite，beta 版本**

<img src="https://cdn.uwayfly.com/article/2026/1/own_mike_cXASHEhzC3r2PWNS.gif" width="400" />

## 3. 支持的 NPM 包

普通包

- press-ui
- press-plus
- press-next
- press-pix

`monorepo` 模式

- plugin-light
- press-components
- pnpm-npm

⚠️ 注意，`monorepo` 模式下发布时必须写明发布哪个子包。
