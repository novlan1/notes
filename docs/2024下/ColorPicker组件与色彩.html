Saturation СИ║жЦ▒тњїт║д№╝їтЈѕуД░УЅ▓т║дТѕќтйЕт║д№╝їУАеуц║жбюУЅ▓ТјЦУ┐ЉтЁЅУ░▒УЅ▓уџёуеІт║д№╝їтЈќтђ╝УїЃтЏ┤СИ║ 0-100%сђѓжЦ▒тњїт║джФўСйјС╝џтй▒тЊЇУЅ▓тйЕуџёТўјУЅ│уеІт║д№╝їтдѓтцДу║бт░▒Т»ћујФу║бТЏ┤у║б№╝їУ┐Ўт░▒Тў»У»┤тцДу║буџёжЦ▒тњїт║дУдЂжФўсђѓж╗ЉУЅ▓сђЂуЎйУЅ▓С╗ЦтЈіуЂ░УЅ▓Тў»РђюУЅ▓т║д=0РђЮуџёжбюУЅ▓№╝їУ┐ЎСИђуѓ╣тЈ»С╗ЦжђџУ┐ЄТІќТІй saturation жЮбТЮ┐№╝їт╣ХУДѓт»ЪСИІТќ╣тђ╝ТЮЦжфїУ»ЂсђѓуЏ┤уЎйуѓ╣уљєУДБ№╝їтЂЄУ«ЙСйаТюЅСИђуДЇу║бУЅ▓жбюТќЎтњїТ░┤№╝їТ░┤тіауџёУХітцџ№╝їу║бУЅ▓С╝џУХіТиА№╝ѕжЦ▒тњїт║дСйј№╝Ѕ№╝ЏТ░┤тіауџёт░Љ№╝їу║бУЅ▓т░▒С╝џТхЊ№╝ѕжЦ▒тњїт║джФў№╝Ѕсђѓ

Photoshop СИГУ░ЃУіѓУЅ▓т║дуџёТЋѕТъю№╝ЏтидСИіУДњСИ║тјЪтДІтй▒тЃЈ(ТЮЦУЄфу╗┤тЪ║уЎЙуДЉ)сђѓ

Value СИ║Тўјт║д№╝їжбюУЅ▓уџёТўјС║«уеІт║д№╝їтЈќтђ╝УїЃтЏ┤СйЇ 0-100%сђѓТўјт║дТў»СИјтљїТаиС║«уџёуЎйУЅ▓уЅЕСйЊуЏИТ»ћ№╝їТЪљуЅЕуџёС║«уџёуеІт║дсђѓуј░т«ъућЪТ┤╗СИГу┤аТЈЈСйютЊЂт░▒Тў»Тўјт║дуџёУЅ║Тю»№╝їу┤аТЈЈСйютЊЂТ▓АТюЅС╗╗СйЋжЦ▒тњїт║дтњїУЅ▓уЏИт▒ъТђДсђѓСйєжђџУ┐ЄТўјт║дтЇ┤УЃйт░єС║ІуЅЕуџёу╗єУіѓт▒Ћжю▓ТЌажЂЌсђѓ

2. RGB жбюУЅ▓ТеАтъІ РђІ

RGB жбюУЅ▓ТеАтъІТў»ТѕЉС╗гуєЪуЪЦуџёжбюУЅ▓ТеАтъІ№╝їжђџУ┐ЄСИЅтјЪУЅ▓№╝ѕу║бу╗┐УЊЮ№╝ЅС╗ЦСИЇтљїуџёТ»ћСЙІуЏИтіа№╝їтЈ»С╗ЦтЙЌтѕ░тљёуДЇУЅ▓тйЕсђѓ

RGBжбюУЅ▓ТеАтъІТўат░ётѕ░СИђСИфуФІТќ╣СйЊСИісђѓТ░┤т╣│уџёxУй┤С╗БУАеу║бУЅ▓№╝їтљЉтидтбътіасђѓyУй┤С╗БУАеУЊЮУЅ▓№╝їтљЉтЈ│СИІТќ╣тљЉтбътіасђѓуФќуЏ┤уџёzУй┤С╗БУАеу╗┐УЅ▓№╝їтљЉСИітбътіасђѓтјЪуѓ╣С╗БУАеж╗ЉУЅ▓№╝їжЂ«ТїАтюеуФІТќ╣СйЊУЃїжЮбсђѓ

3. HSV тњї HSL жбюУЅ▓ТеАтъІ РђІ

HSL тњї HSV жЃйТў»т░є RGB УЅ▓тйЕТеАтъІСИГуџёуѓ╣УАеуц║тюетюєТЪ▒тЮљТаЄу│╗СИГуџёТќ╣Т│ЋсђѓУ┐ЎСИцуДЇУАеуц║Т│ЋУ»ЋтЏЙтЂџтѕ░Т»ћтЪ║С║југЏтЇАт░ћтЮљТаЄу│╗тЄаСйЋу╗ЊТъёуџё RGB ТеАтъІТЏ┤тіауЏ┤УДѓсђѓ

  • HSL тЇ│УЅ▓уЏИсђЂжЦ▒тњїт║дсђЂС║«т║д№╝ѕУІ▒У»Г№╝џHue, Saturation, Lightness№╝Ѕсђѓ

  • HSV тЇ│УЅ▓уЏИсђЂжЦ▒тњїт║дсђЂТўјт║д№╝ѕУІ▒У»Г№╝џHue, Saturation, Value№╝Ѕ№╝їтЈѕуД░ HSB№╝їтЁХСИГ B тЇ│УІ▒У»Г№╝џBrightnessсђѓ

HSL№╝ѕa~d№╝ЅтњїHSV№╝ѕe~h№╝ЅсђѓСИітЇіжЃетѕє№╝ѕaсђЂe№╝Ѕ№╝џСИцУђЁуџё3DТеАтъІТѕфжЮбсђѓСИІтЇіжЃетѕє№╝џт░єТеАтъІСИГСИЅСИфтЈѓТЋ░уџётЁХСИГС╣ІСИђтЏ║т«џСИ║тИИжЄЈ№╝їтЁХт«ЃСИцСИфтЈѓТЋ░уџётЏЙтЃЈсђѓ

4. HSV тњї HSL уџёСИЇтљї РђІ

тюетјЪуљєтњїУАеуј░СИі№╝їHSL тњї HSV СИГуџё H№╝ѕУЅ▓уЏИ№╝Ѕ т«їтЁеСИђУЄ┤№╝їСйєС║їУђЁуџё S№╝ѕжЦ▒тњїт║д№╝ЅСИЇСИђТаи№╝ї L тњї V №╝ѕТўјт║д №╝ЅС╣ЪСИЇСИђТаи№╝џ

  • HSV СИГуџё S ТјДтѕХу║»УЅ▓СИГТиитЁЦуЎйУЅ▓уџёжЄЈ№╝їтђ╝УХітцД№╝їуЎйУЅ▓УХіт░Љ№╝їжбюУЅ▓УХіу║»№╝Џ
  • HSV СИГуџё V ТјДтѕХу║»УЅ▓СИГТиитЁЦж╗ЉУЅ▓уџёжЄЈ№╝їтђ╝УХітцД№╝їж╗ЉУЅ▓УХіт░Љ№╝їТўјт║дУХіжФў
  • HSL СИГуџё S тњїж╗ЉуЎйТ▓АТюЅтЁ│у│╗№╝їжЦ▒тњїт║дСИЇТјДтѕХжбюУЅ▓СИГТиитЁЦж╗ЉуЎйуџётцџт»А№╝Џ
  • HSL СИГуџё L ТјДтѕХу║»УЅ▓СИГуџёТиитЁЦуџёж╗ЉуЎйСИцуДЇжбюУЅ▓сђѓ

С║їУђЁтюеТЋ░тГдСИіжЃйТў»тюєТЪ▒№╝їСйє HSV№╝ѕУЅ▓уЏИсђЂжЦ▒тњїт║дсђЂТўјт║д№╝ЅтюеТдѓт┐хСИітЈ»С╗ЦУбФУ«цСИ║Тў»жбюУЅ▓уџётђњтюєжћЦСйЊ№╝ѕж╗Љуѓ╣тюеСИІжАХуѓ╣№╝їуЎйУЅ▓тюеСИіт║ЋжЮбтюєт┐Ѓ№╝Ѕ№╝їHSLтюеТдѓт┐хСИіУАеуц║С║єСИђСИфтЈїтюєжћЦСйЊтњїтюєуљЃСйЊ№╝ѕуЎйУЅ▓тюеСИіжАХуѓ╣№╝їж╗ЉУЅ▓тюеСИІжАХуѓ╣№╝їТюђтцДТефтѕЄжЮбуџётюєт┐ЃТў»тЇіуеІуЂ░УЅ▓№╝ЅсђѓТ│еТёЈт░йу«Атюе HSL тњї HSV СИГРђюУЅ▓уЏИРђЮТїЄуД░уЏИтљїуџёТђДУ┤е№╝їт«ЃС╗гуџёРђюжЦ▒тњїт║дРђЮуџёт«џС╣ЅТў»ТўјТўЙСИЇтљїуџёсђѓ

т«ъжЎЁт║ћућетю║ТЎ»СИі№╝їТІЙУЅ▓тЎетцДтцџућеуџёТў» HSV ТеАт╝Ј№╝їТ»ћтдѓ Photoshop тњїтљёу▒╗уџётЅЇуФ»у╗ёС╗Хсђѓ

уюІ Photoshop уџё HSV жбюУЅ▓ТеАтъІТІЙУЅ▓тЎе№╝їтдѓСИІтЏЙТЅђуц║№╝їHSV уџё V№╝ѕТўјт║д№╝ЅТјДтѕХу║»УЅ▓СИГТиитЁЦж╗ЉУЅ▓уџёжЄЈ№╝їУХітЙђСИі№╝їтђ╝УХітцД№╝їж╗ЉУЅ▓УХіт░Љ№╝їжбюУЅ▓Тўјт║дУХіжФўсђѓ

HSV уџё S№╝ѕжЦ▒тњїт║д№╝ЅТјДтѕХу║»УЅ▓СИГТиитЁЦуЎйУЅ▓уџёжЄЈ№╝їУХітЙђтЈ│№╝їтђ╝УХітцД№╝їуЎйУЅ▓УХіт░Љ№╝їжбюУЅ▓у║»т║дУХіжФўсђѓ

ТјЦСИІТЮЦуюІ Affinity Designer уџё HSL жбюУЅ▓ТеАтъІТІЙУЅ▓тЎесђѓтдѓСИІтЏЙТЅђуц║№╝їY Уй┤Тўјт║дУй┤№╝їС╗јСИІУЄ│СИі№╝їТиитЁЦуџёж╗ЉУЅ▓жђљТИљтЄЈт░Љ№╝їуЏ┤тѕ░ 50% СйЇуй«тцёт«їтЁеТ▓АТюЅж╗ЉУЅ▓№╝їС╣ЪТ▓АТюЅуЎйУЅ▓№╝їу║»т║дУЙЙтѕ░ТюђжФўсђѓу╗Ду╗ГтЙђСИіУх░№╝їу║»УЅ▓ТиитЁЦуџёуЎйУЅ▓жђљТИљтбътіа№╝їтѕ░УЙЙТюђжФўуѓ╣тЈўСИ║у║»уЎйУЅ▓№╝їТўјт║дТюђжФўсђѓ

RGB ТеАт╝ЈТЏ┤жђѓтљѕућеС║јтиЦСИџ№╝їHSV тњї HSL ТЏ┤уЏ┤УДѓ№╝їугдтљѕуЏ┤УДЅсђѓ

ТѕЉС╗гуюІтѕ░уџёУ»ЋУЅ▓тЎет░▒Тў» HSV тюєжћЦСИђтЇіуџёТефТѕфжЮб№╝їтЈфСИЇУ┐Єж╗ЉУЅ▓ућ▒СИђСИфуѓ╣ТЅЕт▒ЋТѕљС║єСИђТЮАу║┐сђѓ
