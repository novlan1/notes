Saturation дёәйҘұе’ҢеәҰпјҢеҸҲз§°иүІеәҰжҲ–еҪ©еәҰпјҢиЎЁзӨәйўңиүІжҺҘиҝ‘е…үи°ұиүІзҡ„зЁӢеәҰпјҢеҸ–еҖјиҢғеӣҙдёә 0-100%гҖӮйҘұе’ҢеәҰй«ҳдҪҺдјҡеҪұе“ҚиүІеҪ©зҡ„жҳҺиүізЁӢеәҰпјҢеҰӮеӨ§зәўе°ұжҜ”зҺ«зәўжӣҙзәўпјҢиҝҷе°ұжҳҜиҜҙеӨ§зәўзҡ„йҘұе’ҢеәҰиҰҒй«ҳгҖӮй»‘иүІгҖҒзҷҪиүІд»ҘеҸҠзҒ°иүІжҳҜвҖңиүІеәҰ=0вҖқзҡ„йўңиүІпјҢиҝҷдёҖзӮ№еҸҜд»ҘйҖҡиҝҮжӢ–жӢҪ saturation йқўжқҝпјҢе№¶и§ӮеҜҹдёӢж–№еҖјжқҘйӘҢиҜҒгҖӮзӣҙзҷҪзӮ№зҗҶи§ЈпјҢеҒҮи®ҫдҪ жңүдёҖз§ҚзәўиүІйўңж–ҷе’Ңж°ҙпјҢж°ҙеҠ зҡ„и¶ҠеӨҡпјҢзәўиүІдјҡи¶Ҡж·ЎпјҲйҘұе’ҢеәҰдҪҺпјүпјӣж°ҙеҠ зҡ„е°‘пјҢзәўиүІе°ұдјҡжө“пјҲйҘұе’ҢеәҰй«ҳпјүгҖӮ

Photoshop дёӯи°ғиҠӮиүІеәҰзҡ„ж•Ҳжһңпјӣе·ҰдёҠи§’дёәеҺҹе§ӢеҪұеғҸ(жқҘиҮӘз»ҙеҹәзҷҫз§‘)гҖӮ

Value дёәжҳҺеәҰпјҢйўңиүІзҡ„жҳҺдә®зЁӢеәҰпјҢеҸ–еҖјиҢғеӣҙдҪҚ 0-100%гҖӮжҳҺеәҰжҳҜдёҺеҗҢж ·дә®зҡ„зҷҪиүІзү©дҪ“зӣёжҜ”пјҢжҹҗзү©зҡ„дә®зҡ„зЁӢеәҰгҖӮзҺ°е®һз”ҹжҙ»дёӯзҙ жҸҸдҪңе“Ғе°ұжҳҜжҳҺеәҰзҡ„иүәжңҜпјҢзҙ жҸҸдҪңе“ҒжІЎжңүд»»дҪ•йҘұе’ҢеәҰе’ҢиүІзӣёеұһжҖ§гҖӮдҪҶйҖҡиҝҮжҳҺеәҰеҚҙиғҪе°ҶдәӢзү©зҡ„з»ҶиҠӮеұ•йңІж— йҒ—гҖӮ

2. RGB йўңиүІжЁЎеһӢ вҖӢ

RGB йўңиүІжЁЎеһӢжҳҜжҲ‘д»¬зҶҹзҹҘзҡ„йўңиүІжЁЎеһӢпјҢйҖҡиҝҮдёүеҺҹиүІпјҲзәўз»ҝи“қпјүд»ҘдёҚеҗҢзҡ„жҜ”дҫӢзӣёеҠ пјҢеҸҜд»Ҙеҫ—еҲ°еҗ„з§ҚиүІеҪ©гҖӮ

RGBйўңиүІжЁЎеһӢжҳ е°„еҲ°дёҖдёӘз«Ӣж–№дҪ“дёҠгҖӮж°ҙе№ізҡ„xиҪҙд»ЈиЎЁзәўиүІпјҢеҗ‘е·ҰеўһеҠ гҖӮyиҪҙд»ЈиЎЁи“қиүІпјҢеҗ‘еҸідёӢж–№еҗ‘еўһеҠ гҖӮз«–зӣҙзҡ„zиҪҙд»ЈиЎЁз»ҝиүІпјҢеҗ‘дёҠеўһеҠ гҖӮеҺҹзӮ№д»ЈиЎЁй»‘иүІпјҢйҒ®жҢЎеңЁз«Ӣж–№дҪ“иғҢйқўгҖӮ

3. HSV е’Ң HSL йўңиүІжЁЎеһӢ вҖӢ

HSL е’Ң HSV йғҪжҳҜе°Ҷ RGB иүІеҪ©жЁЎеһӢдёӯзҡ„зӮ№иЎЁзӨәеңЁеңҶжҹұеқҗж Үзі»дёӯзҡ„ж–№жі•гҖӮиҝҷдёӨз§ҚиЎЁзӨәжі•иҜ•еӣҫеҒҡеҲ°жҜ”еҹәдәҺз¬ӣеҚЎе°”еқҗж Үзі»еҮ дҪ•з»“жһ„зҡ„ RGB жЁЎеһӢжӣҙеҠ зӣҙи§ӮгҖӮ

  • HSL еҚіиүІзӣёгҖҒйҘұе’ҢеәҰгҖҒдә®еәҰпјҲиӢұиҜӯпјҡHue, Saturation, LightnessпјүгҖӮ

  • HSV еҚіиүІзӣёгҖҒйҘұе’ҢеәҰгҖҒжҳҺеәҰпјҲиӢұиҜӯпјҡHue, Saturation, ValueпјүпјҢеҸҲз§° HSBпјҢе…¶дёӯ B еҚіиӢұиҜӯпјҡBrightnessгҖӮ

HSLпјҲa~dпјүе’ҢHSVпјҲe~hпјүгҖӮдёҠеҚҠйғЁеҲҶпјҲaгҖҒeпјүпјҡдёӨиҖ…зҡ„3DжЁЎеһӢжҲӘйқўгҖӮдёӢеҚҠйғЁеҲҶпјҡе°ҶжЁЎеһӢдёӯдёүдёӘеҸӮж•°зҡ„е…¶дёӯд№ӢдёҖеӣәе®ҡдёәеёёйҮҸпјҢе…¶е®ғдёӨдёӘеҸӮж•°зҡ„еӣҫеғҸгҖӮ

4. HSV е’Ң HSL зҡ„дёҚеҗҢ вҖӢ

еңЁеҺҹзҗҶе’ҢиЎЁзҺ°дёҠпјҢHSL е’Ң HSV дёӯзҡ„ HпјҲиүІзӣёпјү е®Ңе…ЁдёҖиҮҙпјҢдҪҶдәҢиҖ…зҡ„ SпјҲйҘұе’ҢеәҰпјүдёҚдёҖж ·пјҢ L е’Ң V пјҲжҳҺеәҰ пјүд№ҹдёҚдёҖж ·пјҡ

  • HSV дёӯзҡ„ S жҺ§еҲ¶зәҜиүІдёӯж··е…ҘзҷҪиүІзҡ„йҮҸпјҢеҖји¶ҠеӨ§пјҢзҷҪиүІи¶Ҡе°‘пјҢйўңиүІи¶ҠзәҜпјӣ
  • HSV дёӯзҡ„ V жҺ§еҲ¶зәҜиүІдёӯж··е…Ҙй»‘иүІзҡ„йҮҸпјҢеҖји¶ҠеӨ§пјҢй»‘иүІи¶Ҡе°‘пјҢжҳҺеәҰи¶Ҡй«ҳ
  • HSL дёӯзҡ„ S е’Ңй»‘зҷҪжІЎжңүе…ізі»пјҢйҘұе’ҢеәҰдёҚжҺ§еҲ¶йўңиүІдёӯж··е…Ҙй»‘зҷҪзҡ„еӨҡеҜЎпјӣ
  • HSL дёӯзҡ„ L жҺ§еҲ¶зәҜиүІдёӯзҡ„ж··е…Ҙзҡ„й»‘зҷҪдёӨз§ҚйўңиүІгҖӮ

дәҢиҖ…еңЁж•°еӯҰдёҠйғҪжҳҜеңҶжҹұпјҢдҪҶ HSVпјҲиүІзӣёгҖҒйҘұе’ҢеәҰгҖҒжҳҺеәҰпјүеңЁжҰӮеҝөдёҠеҸҜд»Ҙиў«и®ӨдёәжҳҜйўңиүІзҡ„еҖ’еңҶй”ҘдҪ“пјҲй»‘зӮ№еңЁдёӢйЎ¶зӮ№пјҢзҷҪиүІеңЁдёҠеә•йқўеңҶеҝғпјүпјҢHSLеңЁжҰӮеҝөдёҠиЎЁзӨәдәҶдёҖдёӘеҸҢеңҶй”ҘдҪ“е’ҢеңҶзҗғдҪ“пјҲзҷҪиүІеңЁдёҠйЎ¶зӮ№пјҢй»‘иүІеңЁдёӢйЎ¶зӮ№пјҢжңҖеӨ§жЁӘеҲҮйқўзҡ„еңҶеҝғжҳҜеҚҠзЁӢзҒ°иүІпјүгҖӮжіЁж„Ҹе°Ҫз®ЎеңЁ HSL е’Ң HSV дёӯвҖңиүІзӣёвҖқжҢҮз§°зӣёеҗҢзҡ„жҖ§иҙЁпјҢе®ғд»¬зҡ„вҖңйҘұе’ҢеәҰвҖқзҡ„е®ҡд№үжҳҜжҳҺжҳҫдёҚеҗҢзҡ„гҖӮ

дёӢеӣҫжҳҜHSV:

дёӢдёӨеӣҫжҳҜHSL:

е®һйҷ…еә”з”ЁеңәжҷҜдёҠпјҢжӢҫиүІеҷЁеӨ§еӨҡз”Ёзҡ„жҳҜ HSV жЁЎејҸпјҢжҜ”еҰӮ Photoshop е’Ңеҗ„зұ»зҡ„еүҚз«Ҝз»„д»¶гҖӮ

зңӢ Photoshop зҡ„ HSV йўңиүІжЁЎеһӢжӢҫиүІеҷЁпјҢеҰӮдёӢеӣҫжүҖзӨәпјҢHSV зҡ„ VпјҲжҳҺеәҰпјүжҺ§еҲ¶зәҜиүІдёӯж··е…Ҙй»‘иүІзҡ„йҮҸпјҢи¶ҠеҫҖдёҠпјҢеҖји¶ҠеӨ§пјҢй»‘иүІи¶Ҡе°‘пјҢйўңиүІжҳҺеәҰи¶Ҡй«ҳгҖӮ

HSV зҡ„ SпјҲйҘұе’ҢеәҰпјүжҺ§еҲ¶зәҜиүІдёӯж··е…ҘзҷҪиүІзҡ„йҮҸпјҢи¶ҠеҫҖеҸіпјҢеҖји¶ҠеӨ§пјҢзҷҪиүІи¶Ҡе°‘пјҢйўңиүІзәҜеәҰи¶Ҡй«ҳгҖӮ

жҺҘдёӢжқҘзңӢ Affinity Designer зҡ„ HSL йўңиүІжЁЎеһӢжӢҫиүІеҷЁгҖӮеҰӮдёӢеӣҫжүҖзӨәпјҢY иҪҙжҳҺеәҰиҪҙпјҢд»ҺдёӢиҮідёҠпјҢж··е…Ҙзҡ„й»‘иүІйҖҗжёҗеҮҸе°‘пјҢзӣҙеҲ° 50% дҪҚзҪ®еӨ„е®Ңе…ЁжІЎжңүй»‘иүІпјҢд№ҹжІЎжңүзҷҪиүІпјҢзәҜеәҰиҫҫеҲ°жңҖй«ҳгҖӮз»§з»ӯеҫҖдёҠиө°пјҢзәҜиүІж··е…Ҙзҡ„зҷҪиүІйҖҗжёҗеўһеҠ пјҢеҲ°иҫҫжңҖй«ҳзӮ№еҸҳдёәзәҜзҷҪиүІпјҢжҳҺеәҰжңҖй«ҳгҖӮ

RGB жЁЎејҸжӣҙйҖӮеҗҲз”ЁдәҺе·ҘдёҡпјҢHSV е’Ң HSL жӣҙзӣҙи§ӮпјҢз¬ҰеҗҲзӣҙи§үгҖӮ

жҲ‘д»¬зңӢеҲ°зҡ„иҜ•иүІеҷЁе°ұжҳҜ HSV еңҶй”ҘдёҖеҚҠзҡ„жЁӘжҲӘйқўпјҢеҸӘдёҚиҝҮй»‘иүІз”ұдёҖдёӘзӮ№жү©еұ•жҲҗдәҶдёҖжқЎзәҝгҖӮ
