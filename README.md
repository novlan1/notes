# 个人笔记

依赖安装

```bash
pip3 install mkdocs

pip3 install mkdocs-material

pip3 install mkdocs-literate-nav
```

相关命令

```bash
# 开发
mkdocs serve

# 打包
mkdocs build
```

`python` 虚拟环境

```bash
# 创建虚拟环境
python -m venv venv

# 激活
source venv/bin/activate
```
